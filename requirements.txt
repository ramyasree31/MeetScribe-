# ==============================================================================
# Meeting Bot — Python Dependencies
# ==============================================================================
# This file consolidates all Python dependencies across the monorepo.
# Each service also has its own requirements.txt for isolated installs:
#   - apps/ai-summarizer/requirements.txt
#   - apps/audio-processor/requirements.txt
#
# ------------------------------------------------------------------------------
# HOW TO INSTALL
# ------------------------------------------------------------------------------
#
# OPTION 1 — Install ALL Python deps at once (root level):
#
#   python3 -m venv .venv              # create a virtual environment
#   source .venv/bin/activate          # activate it (macOS/Linux)
#   # OR on Windows: .venv\Scripts\activate
#   pip install -r requirements.txt    # install everything
#
# OPTION 2 — Install per-service (recommended for Docker / CI):
#
#   # For the AI Summarizer only:
#   cd apps/ai-summarizer
#   python3 -m venv .venv && source .venv/bin/activate
#   pip install -r requirements.txt
#
#   # For the Audio Processor only:
#   cd apps/audio-processor
#   python3 -m venv .venv && source .venv/bin/activate
#   pip install -r requirements.txt
#
# OPTION 3 — Upgrade all packages to their latest versions:
#
#   pip install --upgrade -r requirements.txt
#
# NOTE: Python 3.11+ is recommended. Make sure ffmpeg is installed on your
#       system for the audio-processor's S3 MP3 conversion feature:
#         macOS:  brew install ffmpeg
#         Ubuntu: sudo apt-get install ffmpeg
# ==============================================================================
